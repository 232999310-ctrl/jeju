## 四、其他奢侈品（Gucci / Prada / Tiffany / Bvlgari等）

### 🔑 返点团号是关键！

**SG模式**（适合品牌杂）：
- 按总消费金额返点15-30%
- 可叠加金卡折扣
- 操作简单，小白首选

**MG模式**（适合买高返点品牌）：
- 按品牌分别计算返点率
- 部分品牌可达30%+
- 不能和金卡同时用

**高返点特殊团号**（⚠️血泪教训！）：
- 同一品牌不同团号返点率可能差2-3倍！
- 如：Bvlgari 普通团4% vs 高返点团10%
- **买奢侈品前一定先问返点公司：这个品牌有没有特殊团号？**

### 操作流程：
```
① 提前1-3天 → 淘宝/微信找「韩国免税店返点」→ 发护照+机票 → 拿团号
② 到了免税店 → 先办金卡（乐天12楼/新罗1楼，免费）
③ 购物结账 → 报团号 + 出示金卡 → 检查小票团号正确！
④ 小票拍照 → 发给返点公司客服
⑤ 离境后 → 1-3个工作日返现到账
```

### ⚠️ 避坑：
- 结账后**当场检查小票**有无团号，无团号→立刻退货重结！
- 不要向柜姐问返点问题（会被投诉）
- 不要用第三方免费接送机（会被偷挂团号）
- 一旦商品送到机场，**不能改团号！不能退换！**

## 六、退税攻略 💰

### 三种退税方式对比：

| 方式 | 哪里用 | 怎么操作 |
|------|--------|----------|
| **即时退税** | OY/便利店/有TaxFree标志的店 | 买单时出示护照→店员自动退税→支付税后价 |
| **免税店** | 乐天/新罗 | 已经是免税价→付钱→机场提货 |
| **机场退税** | 独立小店(无即时退税) | 买单时拿退税单→机场KIOSK机扫描→安检后退钱 |

### 🔑 退税关键选择：
- **选韩元现金！选韩元现金！选韩元现金！**
- ❌ 选人民币：₩30,000 → 只退到¥110（汇率差很大）
- ✅ 选韩元：₩30,000 → 拿到₩30,000 → 在机场花掉！

### 退税流程：
```
① 安检前 → 3号门附近找蓝色KIOSK退税机 → 扫护照+退税小票
② 安检后 → 人工退税柜台 → 选韩元现金
③ 28号登机口旁也有退税机
```

### ⚠️ 注意事项：
- 便利店满₩15,000也能退税！（很多人不知道）
- 机场OY买的面膜/湿敷棉片 → **必须托运！**不要挑战运气
- 5号登机口机器是国内出发用的，别排错
